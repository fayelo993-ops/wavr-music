# WAVR Music App

WAVR Music est une application web qui permet de rechercher et écouter des extraits de musique grâce à l'API Deezer.

## Fonctionnalités

- Recherche de musique
- Navigation par genre
- Lecteur audio
- Interface moderne

## Technologies utilisées

- HTML
- CSS
- JavaScript
- API Deezer
